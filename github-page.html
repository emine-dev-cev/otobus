🚌 Bus Ticket Reservation System

Python FastAPI tabanlı, tam özellikli mikroservis otobüs bilet rezervasyon sistemi. PostgreSQL, Redis, Kafka, Docker ve Kubernetes ile production-ready mimari.

🐍 Python 3.11 🐳 Docker ☸️ Kubernetes 📨 Apache Kafka 🐘 PostgreSQL 15 ⚡ Redis 7

🚀 Hızlı Başlangıç

git clone https://github.com/emine-dev-cev/bus-ticket-app.git
cd bus-ticket-app
docker-compose up --build
